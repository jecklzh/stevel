记录下怎么安装Hexo

一、安装环境
---
1.安装: [Node.js](https://nodejs.org/en)

2.安装: [Git](https://git-scm.com/downloads)

二、安装Hexo
---
1.运行命令:
```bash
npm install -g hexo-cli
```
2.新建一个博客文件夹，然后cd到这个文件夹，运行:
```bash
hexo init
```
